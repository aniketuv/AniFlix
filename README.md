# AniFlix - The Ultimate Open Source IPTV Player

![AniFlix Banner](https://via.placeholder.com/800x200/0f0f0f/e50914?text=ANIFLIX)

**AniFlix** is a modern, web-based IPTV player capable of streaming thousands of free global TV channels directly in your browser. It features a Netflix-inspired dark UI, PWA support (installable as an app), and powerful filtering capabilities.

## üöÄ Features

* **Global Access:** Browse channels by Category, Country, Language, or Region.
* **Zero Setup:** No registration or subscriptions required. Just load and play.
* **HLS Support:** Built-in HLS.js player for smooth streaming.
* **PWA Ready:** Installable on Android, iOS, Windows, and macOS as a native app.
* **Responsive:** Works perfectly on Mobile, Tablet, and Desktop.
* **Search:** Instantly filter channels within the loaded playlist.
* **Unrestricted:** No content filters‚Äîaccess the full database.

* **HTML5 / CSS3 / Vanilla JavaScript** (No frameworks required)

## üì¶ Installation & Deployment

You can host this project for free using **GitHub Pages**.

1.  **Fork or Clone** this repository.
2.  Go to your repository **Settings**.
3.  Navigate to **Pages** (under the "Code and automation" section).
4.  Select the **main** branch as the source.
5.  Click **Save**.
6.  Your app will be live at `https://<your-username>.github.io/<repo-name>/`
